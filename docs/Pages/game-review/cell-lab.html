Обзор игры Cell Lab
Дата обзора: 02.08.2023

Cell Lab это симулятор малолеточных организмов для мобильных платформ.

Для полного прохождения игры вам нужно пройти 45 испытаний.
В каждом испытании есть окружающая среда и условия прохождения,
а игроку нужно создать организм который выполнит все условия.

Организм состоит из 20 так называемых "режимов",
каждый из которых можно настроить, что дает большой простор для фантазии.

В игре еще можно создать свою среду для экспериментов.
В которой например можно создать свою экосистему.

Разработчик планирует выпустить игру на ПК.
Он ведет блог на своем сайте.
